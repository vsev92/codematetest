## Download project
```bash
git clone git@github.com:vsev92/codematetest.git
cd  codematetest
```
## Install project
```bash
make install
```

## Run tests in memory (with SQLite)
```bash
make test
```
## deploy with docker localy
```bash
make dockerUp
```
then service is avaible on 127.0.0.1:8000
